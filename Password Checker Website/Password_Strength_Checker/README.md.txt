Analyse a dataset of passwords to evaluate their strength and provide insights on common patterns in weak and strong passwords. This will be presented on a webapp whereby a entry tool is provided for users to test their password strengths. Optionally, I can build a password strength metre based on the analysis.

# Project Structure:

This project is organized into the following folders:

- **data**  
  Contains datasets and any raw data files used for analysis or processing.

- **src**  
  Includes the core source code files, including the main logic and functionality of the application.

- **static**  
  Holds static files such as CSS stylesheets, JavaScript files, and images used by the website.

- **templates**  
  Contains HTML templates that are rendered by the Flask application to create the web pages.

- **tests**  
  Stores test scripts and files used to validate the functionality of the project components.
